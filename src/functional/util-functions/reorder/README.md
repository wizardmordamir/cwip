## reorder
  - function that will reorder an array based on given from and to index positions.

### Arguments
  - `array (Array)`: The array to process
  - `fromIndex (Number)`: The index to move element from
  - `toIndex (Number)`: The index to move element to

```js
reorder(array: any[], fromIndex: number, toIndex: number)

// Examples
const array = [1, 2, 3, 4, 5];

reorder(array, 0, 2) // [2, 3, 1, 4, 5]
  ```
